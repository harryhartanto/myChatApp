This is sample of chat application with Server and Client with Java based.
In order to run, you may need to install Java in you machine.
Please find the installer in the url below
https://drive.google.com/open?id=0BxjVbF9aGlqOOGF4WURlTjJRZDQ

----------------------------------------------------------------------------------------------------------------------
Please do steps below if you haven't install java in your machine
----------------------------------------------------------------------------------------------------------------------
1. Execute the installer of Java.
2. Once done, you may need to set the environment variable as below
	a. go to Control Panel\System and Security\System
	b. Click Advanced System Setting
	c. Click Environment Variables
	d. in the System variables (below section) search for Path and click Edit
	e. Insert the directory where you install you Java, sample as below
	ex: C:\Program Files (x86)\Java\jdk1.5.0_12\bin;
----------------------------------------------------------------------------------------------------------------------



to run the application, you may open Server.jar and Client.jar to execute the application.


----------------------------------------------------------------------------------------------------------------------
							Server
----------------------------------------------------------------------------------------------------------------------
Server only contains 1 button to start the app, you may choose the port of server that you want to start.
the IP will automatically using the localhost of the machine that run the Server.

----------------------------------------------------------------------------------------------------------------------


----------------------------------------------------------------------------------------------------------------------
							Client
----------------------------------------------------------------------------------------------------------------------
To demonstrate you may need more than 1 client to run, you may run the Client.jar many times.

1. Insert username, Server and the port details, click Connect to start the app
2. The lobby is the chat room for everyone to say, it is a broadcast room.
3. Once the others user connect, the list of Parties on the right side will be updated. 
Double Click it to open new chat room, that is a private room to chat with the others.
4. To send the message, you may click Enter.
